**1. `weekly-focus-widget.tsx`:**
Props: `{ goals: WeeklyFocusGoal[] }`. Import type from dashboard service.
- Card with header: TargetIcon + "This Week's Focus" title
- Each goal rendered as a row with:
  - Category color dot (6px circle, `backgroundColor: goal.category?.color`)
  - Title (text-sm, truncated with `truncate`)
  - GoalPriorityBadge on the right
  - If goal has progress > 0, show a thin inline progress bar below the title

**2. `progress-overview-widget.tsx`:**
Props: `{ categories: CategoryProgress[] }`. Import type from dashboard service.
- Card with header: BarChart3Icon + "Progress Overview" title
- Each category as a row:
  - Category color dot + name on the left
  - "X/Y" count on the right (completed/total)
  - Below: a full-width bar with Tailwind background (`bg-muted` for track, inline style for fill with `backgroundColor: category.color` and `width: ${percentage}%`)
- Empty state: "Assign goals to categories to see progress breakdown."
- Sort categories by percentage descending so most progress shows first

**3. `streaks-stats-widget.tsx`:**
Props: `{ stats: StatsData }`. Import type from dashboard service.
- Card with header: FlameIcon + "Stats" title
- Grid of stat items (2 columns on all sizes):
  - "Completed this month": `stats.completedThisMonth` with CalendarCheckIcon
  - "Completion rate": `stats.completionRate`% with PercentIcon
  - "Total goals": `stats.totalGoals` with TargetIcon
  - "Total completed": `stats.totalCompleted` with CheckCircle2Icon
- Each stat: icon (size-4, muted foreground), value (text-2xl font-mono font-bold), label (text-xs text-muted-foreground)
- XP/level/streak fields exist in StatsData but are not yet populated (Phase 9). Do NOT display them in this widget. When Phase 9 adds gamification, this widget will be extended with level/XP/streak rows.

**4. `upcoming-deadlines-widget.tsx`:**
Props: `{ goals: DeadlineGoal[] }`. Import type from dashboard service.
- Card with header: CalendarClockIcon + "Upcoming Deadlines" title
- Split goals into two sections:
  - "Next 7 days": goals where deadline <= addDays(now, 7). Use `date-fns` `isAfter`, `isBefore`, `addDays`.
  - "7 to 14 days": remaining goals
- Each goal row:
  - Category color dot + title (truncated)
  - Deadline formatted as "MMM d" via `date-fns` `format`
  - Horizon badge (text-[0.6rem])
- Empty state: "No upcoming deadlines. Your schedule is clear."
- Goals due within 2 days get a `text-destructive` color on the date to signal urgency
