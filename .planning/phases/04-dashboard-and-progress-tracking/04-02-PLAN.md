Implementation:
- Default state: Show a compact `+1` Button (size="sm", variant="outline"). Clicking it calls `useLogProgress().mutateAsync({ goalId, data: { value: 1 } })` and shows `toast.success("+1 {unit} logged")`.

  - On success: toast, close popover, reset inputs
- Disable the button and show a spinner while `isPending` is true on the mutation
