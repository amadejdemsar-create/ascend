## Decisions Made
- **Prisma 7 patterns throughout:** Used `prisma-client` generator (not `prisma-client-js`), `prisma.config.ts` for datasource config (no `url` in schema), and `generated/prisma` output path
- **Standalone Docker output:** Enables minimal production image (~150MB vs ~1GB) with only required files copied
- **Startup migration with graceful skip:** CMD runs `prisma migrate deploy` before `node server.js`, but skips if no migrations directory exists yet (prevents crash on first deploy before schema work is done)
- **Dokploy personal VPS:** Deployed to the same Hostinger VPS used for NativeAI projects, under the "Ascend" project with separate PostgreSQL service

- **Found during:** First deployment

- **Files modified:** Dockerfile
