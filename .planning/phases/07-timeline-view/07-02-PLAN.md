  <name>Task 1: Extract GoalTimelineNode as a memoized component</name>
  <files>components/goals/goal-timeline-node.tsx</files>
  <action>
Completely rewrite `components/goals/goal-timeline-node.tsx`. The old component used Collapsible popups within swim lanes. The new component renders a single gantt row (tree label cell + bar cell) within the parent CSS Grid.

**Important:** This component renders a `React.Fragment` containing TWO sibling divs that are direct children of the parent CSS Grid in GoalTimelineView. It does NOT have its own wrapper div.

1. "use client" directive.

2. Imports:
   - `React` from react
   - `useUIStore` from `@/lib/stores/ui-store`
   - `getGoalColumns` from `@/lib/timeline-utils`
   - `type FlatTimelineRow, type TimeSegment` from `@/lib/timeline-utils`
   - `ChevronRight` from lucide-react
   - `cn` from `@/lib/utils`

3. Props interface:
```typescript
interface GoalTimelineNodeProps {
  row: FlatTimelineRow;
  segments: TimeSegment[];
  timelineYear: number;
  minColWidth: string;
  onToggleExpand: (id: string) => void;
}
```

4. Component function (named `GoalTimelineNodeInner`, exported as `React.memo(GoalTimelineNodeInner)`):
   - Read from store: `const selectGoal = useUIStore((s) => s.selectGoal)` and `const selectedGoalId = useUIStore((s) => s.selectedGoalId)`.
   - Derive: `isSelected = selectedGoalId === row.goal.id`, `isCompleted = row.goal.status === "COMPLETED"`, `isAbandoned = row.goal.status === "ABANDONED"`, `categoryColor = row.goal.category?.color ?? "#6B7280"`.
   - Compute bar position: `const cols = getGoalColumns(row.goal, segments, timelineYear)`.
   - Determine `hasDates = !!row.goal.deadline` (startDate check optional since TreeGoal may not have it populated).

5. Return `<React.Fragment>`:

**Cell 1: Tree label (column 1, sticky left):**
```tsx
<div
  className={cn(
    "sticky left-0 z-10 bg-background border-b border-r flex items-center gap-1.5 py-1.5 text-sm cursor-pointer hover:bg-muted/50 transition-colors",
    isSelected && "bg-primary/5",
  )}
  style={{ paddingLeft: `${row.depth * 1.25 + 0.75}rem` }}
  onClick={() => selectGoal(row.goal.id)}
>
  {row.hasChildren ? (
    <button
      type="button"
      className="shrink-0 rounded p-0.5 hover:bg-muted"
      onClick={(e) => { e.stopPropagation(); onToggleExpand(row.goal.id); }}
    >
      <ChevronRight className={cn("size-3.5 transition-transform duration-150", row.isExpanded && "rotate-90")} />
    </button>
  ) : (
    <span className="w-3.5 shrink-0" />
  )}
  {row.goal.category && (
    <span className="size-2 rounded-full shrink-0" style={{ backgroundColor: categoryColor }} />
  )}
  <span className={cn("truncate font-medium", isCompleted && "line-through text-muted-foreground", isAbandoned && "text-muted-foreground")}>
    {row.goal.title}
  </span>
</div>
```

**Cell 2: Bar area (spans columns 2 through end):**
```tsx
<div
  className="border-b relative"
  style={{
    gridColumn: `2 / ${segments.length + 2}`,
    display: "grid",
    gridTemplateColumns: `repeat(${segments.length}, minmax(${minColWidth}, 1fr))`,
  }}
>
  {cols && (
    <button
      type="button"
      className={cn(
        "relative h-6 my-1 rounded px-1.5 text-[11px] font-medium truncate text-left transition-colors hover:brightness-110",
        isCompleted && "line-through text-muted-foreground",
        isAbandoned && "opacity-50",
      )}
      style={{
        gridColumn: `${cols.start} / ${cols.end}`,
        backgroundColor: `${categoryColor}20`,
        borderLeft: `3px ${hasDates ? "solid" : "dashed"} ${categoryColor}`,
        minWidth: "2rem",
      }}
      onClick={() => selectGoal(row.goal.id)}
    >
      {row.goal.title}
      {row.goal.progress > 0 && (
        <span
          className="absolute inset-y-1 left-0 rounded opacity-20 pointer-events-none"
          style={{
            width: `${Math.min(row.goal.progress, 100)}%`,
            backgroundColor: categoryColor,
          }}
        />
      )}
    </button>
  )}
</div>
```
