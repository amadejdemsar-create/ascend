   Add new state properties to UIStore interface:

   - `activeFilters: ActiveFilters` (default: `{}`)
   - `activeSorting: SortingState` (default: `[]`)
   - `setActiveView: (view: ViewType) => void`
   - `setActiveFilters: (filters: ActiveFilters) => void`
   - `setActiveSorting: (sorting: SortingState) => void`
   - `resetFilters: () => void`

   Update the persist configuration:
   - Add `version: 1` to the persist options.
   - Add a `migrate` function that handles version 0 (the current store without version):

     migrate: (persistedState: unknown, version: number) => {

         return {
           ...(persistedState as Record<string, unknown>),
           activeView: "cards",
           activeFilters: {},
           activeSorting: [],
         };
       }
       return persistedState as Record<string, unknown>;
     },
     ```

     partialize: (state) => ({

     ```
